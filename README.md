# Resumo do curso Azure OpenAI no PlayGround
O curso ensina a utilização do Azure OpenAI no Playground, um ambiente interativo para testar modelos de IA, ajustar parâmetros e criar aplicações multimodais com texto, imagens e áudio. Voltado para iniciantes e profissionais de TI, o curso abrange desde a configuração inicial, que exige conta Azure, permissões administrativas e créditos, até aplicações práticas. Explora o deploy de recursos no Azure, navegação no Azure AI Foundry, tokenização e ajuste de parâmetros como temperatura e Top-P para gerenciar a aleatoriedade, além de penalidades para evitar repetições ou diversificar respostas. Técnicas de prompt engineering são destacadas para desenvolver chats CLI, blobs de dados, imagens via Dall-E e áudio, com foco em clareza e especificidade. Atividades práticas incentivam testes iterativos, documentação de configurações e uso ético de ferramentas de IA, recomendando o Dall-E apenas para prototipagem.
